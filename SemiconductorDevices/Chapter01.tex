\chapter{PN结的静电特性}

在本章中，我们将
\begin{itemize}
    \item 讨论均匀掺杂PN结的结构。
    \item 讨论PN结在热平衡下的能带图和空间电荷区的形成。
    \item 应用泊松方程求解空间电荷区的电场，并计算内建电势差。
    \item 分析反偏时PN结发生的变化，推导空间电荷区宽度和结电容的表达式。
    \item 分析PN结的耐压特性和PN结的击穿。
\end{itemize}
在本章，主要讨论PN结在零偏和反偏时的静电特性，下一章将讨论PN结电流--电压特性。
